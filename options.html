API keys (bring your own)

No keys are embedded in the extension. Each provider is optional except for the features that need it: DeepSeek for summaries and quizzes; Anthropic for the interactive concept graph.

DeepSeek

Get a key at platform.deepseek.com. Saving runs a connection test first.

Anthropic (interactive graph)

Used to build a colored, interactive graph of the current article (nodes, labeled edges). Get a key at console.anthropic.com. Saving runs a connection test first.
